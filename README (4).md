## 🌐 Resume Website

The `index.html` is a fully self-contained, single-file portfolio website.

**Features:**
- Dark terminal aesthetic with neon accent theme
- Smooth scroll + fade-in animations
- Sections: Skills · Experience · Projects · Hackathons · Education · Contact
- Mobile responsive
- Zero dependencies — pure HTML / CSS / JS

**Deploy to GitHub Pages:**

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Your site will be live at `https://<your-username>.github.io/<repo-name>`

## 📝 Word Resume (`.docx`)

The `YuvarajS_Resume.docx` is auto-generated using the `docx` npm package.

**To regenerate the `.docx` after edits:**

```bash
# Install dependency
npm install docx

# Run the generator
node resume_code.js

# Output: YuvarajS_Resume.docx
```

**To edit resume content**, open `resume_code.js` and find the section you want to change.
Each section is clearly commented:

```js
// ── PROJECTS ─────────────────────────────────────────
// ── TECHNICAL SKILLS ─────────────────────────────────
// ── HONORS & ACHIEVEMENTS ────────────────────────────
// ... etc.
```
